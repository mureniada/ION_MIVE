# Source Register

This clean-room pack was synthesized from canonical project materials, not from prior implementation code.

Primary conceptual sources:
- MIVE Foundation Document v1.0
- ION V2 Engineering Rules & Development Invariants v1.0
- ION V2 Architecture Freeze v1.0
- ION_PLUS Continuation Pack v1.0
- ION_PLUS product work plan / target product v1.0

The clean-room pack intentionally retains:
- mission;
- epistemic principles;
- IVE/MIVE meanings;
- evidence rules;
- user output requirements;
- cost instrumentation;
- local-first delivery sequence.

It intentionally removes:
- historical module status;
- Codex-specific instructions;
- previous implementation details;
- debug transcripts;
- path fixes;
- frozen code assumptions;
- claims that an old implementation is authoritative.

The source books and corpus files must be supplied separately by the operator.
